# White_Blood_Catcher
Alternate version of the bioinformatics class mini-project, without any result files and the jupyter file only.

For this code to work, the following packages must be installed:
1. Tensorflow
2. Pandas
3. Matplotlib
4. Numpy
5. Pillow

This is the alternate repository for the bioinformatics mini-project, which was made as a 'blank' version of the porject for those who would like to try running the code from the beginning. The actual required Github repository for this class can be found [here](https://github.com/naomizolasebayang/Bioinformatics_MiniProject_IUP_Group_1.git)
